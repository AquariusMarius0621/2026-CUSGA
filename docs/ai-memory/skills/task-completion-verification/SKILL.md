---
name: task-completion-verification
description: Verify multi-part `【塔防开发】` requests against the user checklist before wrapping up. Use when the request contains multiple independent deliverables such as code + scene + UI + docs.
---

1. Convert the request into a flat checklist.
2. Verify each item with concrete evidence from files, scene state, or validation steps.
3. Mark anything only partially verified.
4. Do not hide gaps behind a generic “done”.
5. In the final response, group results by outcome rather than by file dump.
